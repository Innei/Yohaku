# 开发时的对话记录

做 Yohaku 的时候，我发现这些和 AI 一起写的对话往往比最后那段代码更有用——思路、取舍、踩过的坑都在里面。所以把会话都开源在这里了。

按年份分了文件夹：`2025/`、`2026/`。

---

**English.** While building Yohaku, I found these AI-assisted chats often more useful than the code alone—how choices were made, what failed, what stuck. So I’m sharing them here, grouped by year (`2025/`, `2026/`).

**日本語。** Yohaku を作っているうちに、最終的なコードより、AI と書きながら進めた対話の方が役に立つことが多いと感じました。判断や試行錯誤がそのまま残っているからです。`2025/`、`2026/` に分けて置いています。
